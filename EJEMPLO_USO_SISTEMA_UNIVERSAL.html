Ejemplo de Código para Integrar el Sistema

// Ejemplo de cómo crear una tarea desde cualquier módulo const taskData = { titulo: "Reparar equipo de refrigeración", descripcion: "El equipo presenta fallos en el sistema de enfriamiento", prioridad: "alta", tipo: "correctivo", ubicacion_solicitud: "Cubierta 5 - Cocina Principal", equipo_afectado: "Refrigerador industrial", tiempo_estimado: 2.5, modulo_origen: "restaurante", origen_url: window.location.href }; fetch('/api/tasks/create/', { method: 'POST', headers: { 'Content-Type': 'application/json', 'X-CSRFToken': getCSRFToken() }, body: JSON.stringify(taskData) }) .then(response => response.json()) .then(data => { if (data.success) { console.log('✅ Tarea creada:', data.task_id); showNotification('Tarea enviada a mantenimiento', 'success'); } }) .catch(error => { console.error('❌ Error:', error); showNotification('Error al crear tarea', 'error'); });
Consejo

El botón universal se carga automáticamente cuando incluyes el archivo JavaScript en tu template. Solo necesitas incluir: <script src="{% static 'js/cruise_maintenance_button.js' %}"></script>
